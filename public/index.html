Anki import

AnkiConnect setup
  1. Install the AnkiConnect add-on.
  2. Restart Anki after installing the add-on.
  3. Keep Anki open while importing sentences.
  4. Click Connect Anki, choose a deck, then choose the English and Japanese fields.

Bunpro import

How to find your Bunpro token
  1. Log in to Bunpro in your browser.
  2. Open developer tools with F12.
  3. Open the Network tab.
  4. Refresh Bunpro.
  5. Search for user in the Network tab.
  6. Click the request named user.
  7. Open Headers and scroll down to Request Headers.
  8. Find Authorization and copy the token after Token token=.

Paste only the token value here. Your saved token stays in your local `.env` file.

Browser extensions with page access may read fields while you type. Manual `.env` editing avoids typing keys into the browser.

When enabled, Bunpro grammar points are not repeated until every available grammar point has had one question answered.

CSV file import

CSV format

Export from Excel, Google Sheets, or another spreadsheet app as CSV. The first row should contain column names. Choose one English column, one Japanese column, and optionally a hint column.

English Japanese Hint
The sea is beautiful. 海が綺麗だ。 Use が for the subject.
I went to Tokyo yesterday. 昨日東京に行きました。 Past tense polite form.

Imports are capped at 5,000 rows and 2 MB. Very long fields are shortened before saving.

LLM settings

Choose a provider you are allowed to use: a local OpenAI-compatible server, OpenAI, Gemini, or another compatible Chat Completions API.

Gemini keys are available from Google AI Studio and OpenAI keys from OpenAI Platform. Before using Gemini, check Google's current Gemini API terms, including age, audience, region, billing, and unpaid-quota data-use requirements.

Saved values go to `.env`. Secret values are not shown again after saving.
